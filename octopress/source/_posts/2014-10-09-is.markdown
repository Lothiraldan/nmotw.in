# [is](https://www.npmjs.org/package/is)
>This is a definitive JavaScript type testing library.


__Get it:__ ```npm install is```


__Sample usage:__

```javascript
var is = require('is');
is.type(true, 'boolean');
is.type(1, 'boolean');
is.type(1, 'number');
is.type([], 'object');
is.array([]);
is.odd(9);
is.even(2);
// So on...
```
